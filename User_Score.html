User Score
User Score is a value assigned to each user that measures their overall activity in TaylorBowl. It is similar to Elo for tracks, but measures activity instead of popularity. Like Elo, users are divided into User Tiers based on their User Score. Certain users are also given special roles or titles based on their participation, most notable of which is Sovereign of the Pantheon. User Score is completely reset at the end of each season. You can use the command !score to check your User Score and your measured activity in different areas.
User Score is measured from 10 components, which are Ephemera Condensed, Bowls Started, Bowls Participated, LyricRace IQ, Cleared Floors, First Votes, Hammer Votes, Streak Record, Spent Credits, and Exceeds. These components either contribute to your Base Score or Additional Score, and have different numbers of points each as detailed below. A user's total User Score is the sum of their Base Score and Additional Score. During Season Resets, all of these components reset to 0, which resets all users' User Score. They are still stored in a user's All-Time statistics. Users can view their User Score, activity in each component, and the number of points each component contributes using the command !score user where user is an optional parameter that defaults to the command user.

Base Score
Ephemera Condensed, Bowls Started, Bowls Participated, LyricRace IQ, Cleared Floors, First Votes, and Hammer Votes contribute to Base Score. Users can check their activity in all these components using the command !score.

Ephemera Condensed refers to the number of Ephemera a user has condensed during a Season, which are worth 25 points each. The price of condensing does not affect the number of points. Using !decompress will reduce a user's Ephemera Condensed by 1 for each successful usage. Users can check the users with the 13 highest Ephemera Condensed using the command !rankephemeracondensed or its shorter alternative !rankephemera.

Bowls Started refers to the number of TaylorBowls a user has started, forced, or queued during a Season, which are worth 20 points each. The Tier, Duration, price, etc. of the TaylorBowl does not affect the number of points. Users can check the users with the 13 highest Bowls Started using the command !rankbowlsstarted or its shorter alternative !rankstarted.

Bowls Participated refers to the number of Bowls a user has voted in during a Season, which are worth 5 points each. Autovotes and influenced votes also count as 5 points. Hammer Votes and First Votes count as both Bowls Participated and towards their own categories. Users can check the users with the 13 highest Bowls Participated using the command !rankbowlsparticipated or its shorter alternative !rankbowls.

LyricRace IQ refers to the user's LyricRace IQ in LyricRace, which is worth 1 point per 0.1 IQ, rounded off. Users can check the users with the 13 highest LyricRace IQ using the command !rankiq.

Cleared Floors refers to the number of Floors in The Dojo a user has cleared in total during a Season, which are worth 1 point per 13 Cleared Floors, rounded down. The difficulty of the Floor, the Album fought, etc. do not affect the number of points. Users can check the users with the 13 highest Cleared Floors using the command !rankclearedfloors or its shorter alternative !rankfloors.

First Votes refers to the number of earliest votes in a TaylorBowl, counted across all servers, a user has cast during a Season, which are worth 1 point each. Since they also count as Bowls Participated, each First Vote is worth a total of 6 points. Users can check the users with the 13 highest First Votes using the command !rankfirstvotes or its shorter alternative !rankfirstvote.

Hammer Votes refers to the number of "tiebreaking" votes a user has cast during a season, which are worth 1 point each. In more detail, Hammer Votes are the final vote in a bowl across all servers, if it is for the winning song, and said winning song won by a margin of 1 vote. Since they also count as Bowls Participated, each Hammer Vote is woth a total of 6 points. Users can check the users with the 13 highest Hammer Votes using the command !rankhammervotes or its shorter alternative !rankhammer.

The sum of all points from these 7 categories is a user's Base Score.
Additional Score
Streak Record, Spent Credits, and Exceeds contribute to a user's Additional Score. Each component of Additional Score cannot be greater than a certain percentage of a user's Base Score, and once this value is reached, a Surplus version of that component contributes to a user's Additional Score using a different formula. Users can check their activity in all these components using the command !score.

Streak Record refers to a user's highest Streak during a Season, which is worth 1% Base Score per Streak Record, up to a maximum of 200% Base Score. Surplus Streak Record is worth 0.01% Base Score per Surplus Streak Record. Users can check the users with the 13 highest Streak Records using the command !rankstreakrecords or its shorter alternative !rankstreakrecord.

Spent Credits refers to the number of credits a user has spent in a season, excluding credits lost to minigames and credits used to condense Ephemera, which are worth 1 User Score each, up to a maximum of 100% Base Score. Surplus Spent Credits are worth the 0.6th root of total Surplus Spent Credits in points, or given by the formula:

$$\text{Points}=\left(\text{Surplus Spent Credits}\right)^{0.6}$$

Users can check the users with the 13 highest Spent Credits using the command !rankspentcredits or its shorter alternative !rankspent. Exceeds refers to the number of times a user has used the command !exceed during a Season, which is worth 5% Base Score each, up to a maximum of 100% Base Score. Using the command !crystallize does not reduce Exceeds. Surplus Exceeds are worth 1000 points per Exceed. There is no known way to check the users with the highest Exceeds.

In total, excluding Surplus components, a user's Additional Score can reach a maximum of 400% of their Base Score, which effectively multiplies their Base Score by 5.
Based on their User Score, users will be split into User Tiers and can gain special titles based on their activity. The user with the highest User Score is known as The Sovereign, the top 13 users of User Scores are placed in The Council, and the top 3 users of certain components are placed in The Pantheon. These are collectively known as Accolades. At the end of every Season, each user's highest Accolade is added to their list of Accolades. Users can view their past Accolades by using the command !accolades user where user is an optional parameter that defaults to the command user.
User Tiers
Like tracks Tiers, there are 11 User Tiers from 0 to 10, with the highest being Tier 0. User Tier 0 always has only 13 users, who comprise the Council. Each subsequent Tier is based on the User Score of the 13th highest User Score, or the user with the lowest User Score in Tier 0. The minimum score of Tier 1 is half of the User Score of O-13, the minimum score of Tier 2 is a quarter, etc. For example, if the 13th highest User Score is 100000, then users between 50000 and 100000 are Tier 1. This makes each lower Tier significantly easier to enter as you only need half as much User Score. Users can check their User Tier using the command !stats user or !score user where user is an optional parameter that defaults to the command user.
The Council
The Council is made up of the 13 users with the highest User Scores, and these users are known as Councillors. They are given a title of 'O-X' where X is their rank in the Council. For example, the user with the 4th highest User Score is O-4. The Council and these titles are a reference to the SCP Foundation. Based on SCP lore, the O in O-X stands for Overseer, which is referenced in the footer of the embed showing the Council that reads 'The Council. The Command. The Overseers.' Users can check the members of The Council and their User Scores using the command !council.

The user with the highest User Score is not known as O-1, but instead as The Sovereign of the Pantheon, shortened to The Sovereign, Sovereign, or colloquially 'sov'.

The top 13 users of User Tier 1 are known as Candidates, but they have no special roles or benefits unlike Councillors. Users can check the Candidates by using the command !candidates.
The Pantheon
The Sovereign and the users who are top 3 in the following components are given seats, aka included, in the Pantheon: Bowls Participated, Bowls Started, Streak Record, LyricRace IQ, Ephemera Condensed, Spent Credits, and Floors Cleared. Note that even though First Votes, Hammer Votes, and Exceeds contribute to User Score, they are not given seats in The Pantheon. Users can check the members of The Pantheon and their respective seats by using the command !pantheon.

Each of the included components are known as a different Trinity, referring to the 3 users that comprise the seats of that component. The name of each of these in a user's Accolades also varies between components. Users are also given roles for being in The Pantheon, and their names are shortened versions of their respective Accolade names. This is summarised in the following table:

A user's rank in their respective Pantheon seat also affects their name. Their names based on rank are, in order from 1st to 3rd, Radiant, Ascended, and Attuned. These names are combined with the component name to form the full Accolade name. For example, the 2nd highest user for Bowls Participated is the Ascended Warden of Ubiquity.

Users can only have 1 seat in The Pantheon. If they are eligible for more than 1 position, they are given the seat where they have the highest rank. For example, any seat where the user is 2nd will take precedence over a seat where they are 3rd. If there is more than 1 seat with the same rank, they are given the seat listed earlier in the order used in the above table from top to bottom. For example, a user with the same rank in both Bowls Participated and Spent Credits will be given the seat in Bowls Participated.
Roles & Symbols
Users are given special display roles with symbols if they are in the Council or Pantheon. There is a role for Sovereign, Council, each component in the Pantheon, and the top 3 users of First Votes and Hammer Votes. The Sovereign is classified as a Pantheon role, so a user may not have both Sovereign and a Pantheon component role. However, it is possible for a user to have both Councillor and Sovereign, or Councillor and Pantheon component role. Each of these roles, as well as The Pantheon itself, has an associated symbol taken from astrology, summarised in this table:

Bowls Started Guardian Saturn ♄
Streak Record Regalia Neptune ♆

Spent Credits Quaestor Pluto ⯓
Floors Cleared Vanquisher Ceres ⚳
First Votes Velox Mercury ☿
Hammer Votes Arbiter Chiron ⚷

While most of these symbols are chosen without meaning, the use of Sun for The Sovereign, Moon for The Council, Earth for The Pantheon, and Mercury for Velox were intentional. The Sun, Moon, and Earth represent the hierarchy of the three titles. Mercury was chosen as Velox is for First Votes; Mercury is the first planet in the Solar System and the Roman deity Mercury was associated with speed.

In certain embeds such as those from the commands !council, !pantheon, !candidates, and the rank commands such as !rankbowls, The Sovereign has the Sun before their name and users in The Pantheon have two symbols representing their rank and seat. The rank of users in The Pantheon also have their own associated symbols; Γ for 1st/Radiant, Δ for 2nd/Ascended, Σ for 3rd/Attuned. These symbols were chosen because of the number of strokes it takes to write them, so that The Sovereign, 1st, 2nd, and 3rd places have 1, 2, 3, and 4 strokes representing their hierarchy. For example, the 2nd place user for Streak Record / Ascended Regalia of Resolve will have Δ♆ in front of their name.

All users in The Council and The Pantheon are given access to a private channel called #ministry, and The Sovereign is given an additional private channel called #citadel.
User Score
The calculations for User Score are given by the following formulae. Base Score:

$$\text{Base Score} = 25\cdot\text{Ephemera Condensed}+20\cdot\text{Bowls Started}+5\cdot\text{Bowls Participated}+\bigl\lfloor10\cdot\text{LyricRace IQ}\bigr\rceil+\Biggl\lfloor\cfrac{\text{Cleared Floors}}{13}\Biggr\rfloor+\text{First Votes}+\text{Hammer Votes}$$

Streak Record Additional Score (Surplus Streak Record Score only counts when the user's Streak Record is greater than 200):

$$\text{Streak Record Score} = \min(0.01\cdot\text{Streak Record}, 2)\cdot\text{Base Score} \\ \text{Surplus Streak Record Score} = 0.0001\cdot(\text{Streak Record} - 200)\cdot\text{Base Score}$$

Spent Credits Additional Score (Surplus Spent Credits Score only counts when the user's Spent Credits is greater than their Base Score):

$$\text{Spent Credits Score} = \min(\text{Spent Credits}, \text{Base Score}) \\ \text{Surplus Spent Credits Score} = (\text{Spent Credits} - \text{Base Score})^{0.6}$$

Exceeds Additional Score (Surplus Exceeds Score only counts when the user's Exceeds is greater than 20):

$$\text{Exceeds Score} = \min(0.05\cdot\text{Exceeds}, 1)\cdot\text{Base Score} \\ \text{Surplus Exceeds Score} = 1000\cdot(\text{Exceeds} - 20)$$

Total User Score:

$$\text{User Score}=\text{Base Score}+\text{Streak Record Score}+\text{Surplus Streak Record Score}+\text{Spent Credits Score}+\text{Surplus Spent Credits Score}+\text{Exceeds Score}+\text{Surplus Exceeds Score}$$

User Tiers
The minimum User Score required to enter a given User Tier is given by the formula:

$$\text{Minimum User Score}=\text{O-13 Score}\cdot \left(\cfrac{1}{2}\right)^{\text{User Tier}}$$

$$\text{Streak Record Score} = \min(0.01\cdot\text{Streak Record}, 2)\cdot\text{Base Score} \\ \text{Surplus Streak Record Score} = 0.0001\cdot(\text{Streak Record} - 200)\cdot\text{Base Score}$$

$$\text{Spent Credits Score} = \min(\text{Spent Credits}, \text{Base Score}) \\ \text{Surplus Spent Credits Score} = (\text{Spent Credits} - \text{Base Score})^{0.6}$$

1.4   Exceeds Additional Score (Surplus Exceeds Score only counts when the user's Exceeds is greater than 20):

$$\text{Exceeds Score} = \min(0.05\cdot\text{Exceeds}, 1)\cdot\text{Base Score} \\ \text{Surplus Exceeds Score} = 1000\cdot(\text{Exceeds} - 20)$$

1.5   Total User Score:

$$\text{User Score}=\text{Base Score}+\text{Streak Record Score}+\text{Surplus Streak Record Score}+\text{Spent Credits Score}+\text{Surplus Spent Credits Score}+\text{Exceeds Score}+\text{Surplus Exceeds Score}$$

$$\text{Minimum User Score}=\text{O-13 Score}\cdot \left(\cfrac{1}{2}\right)^{\text{User Tier}}$$
